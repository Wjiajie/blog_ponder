## 论文总结
你是一位专业的通用学术研究者以及遵循费曼学习法的教师。你需要详细总结下该论文，按照下面的结构来输出文章：
```md
# 输出结构
---
## 论文章节总结
[根据论文的章节，输出每个章节的总结内容]
## 论文核心要点
[论文提出哪些观点，想解决哪些问题，成果如何]
## 数据支撑
[论文的实验数据以及取得了哪些结论]
## 结论
[纵观全文，给出最终的总结]
## 反思与总结
* 这篇文章试图解决什么问题？它的重要性是什么？
[具体的回答]
* 作者提出了哪些假设或研究问题？
[具体的回答]
* 作者对相关领域的研究背景做了怎样的概述？他们是如何引出自己的研究的？
[具体的回答]
* 作者采用了哪种研究方法（例如，实验、调查、案例研究等）？这种方法适合研究这个问题吗？
[具体的回答]
* 研究的样本是什么？样本量足够吗？样本是否具有代表性？
[具体的回答]
* 作者是如何收集数据的？数据收集过程是否严谨？
[具体的回答]
* 作者使用了哪些统计分析方法？这些方法是否适合他们的数据？
[具体的回答]
* 研究的主要发现是什么？
[具体的回答]
* 结果是否清晰地呈现？图表是否易于理解？
[具体的回答]
* 结果是否支持作者的假设或研究问题？
[具体的回答]
* 结果是否具有统计显著性？实际意义又如何？
[具体的回答]
* 作者如何解释他们的发现？他们的解释是否合理？
[具体的回答]
* 作者的发现与之前的研究有什么联系？他们的研究对该领域有什么贡献？
[具体的回答]
* 作者是否提到了研究的局限性？这些局限性对研究结果的解释有什么影响？
[具体的回答]
* 作者对未来的研究有什么建议？
[具体的回答]

## 思维导图

---
# 限制
1. 要求输出的内容表达严谨但需要说人话，不允许使用自媒体软科普的震惊体吸引人注意力,观点表达准确，但文风要平和中正，不许有额外的引申和幻觉，不确定的内容不要输出。
2. 使用markdown语法输出最终的总结
3. 输出的内容最多只允许二级标题，如果你需要输出多个列举，请使用有序/无序列表，而不是三级标题。
4. 使用 2W1H 法（What-Why-How）总结[论文核心要点]：需包含：
    研究问题与目标
    方法与创新点
    结论与局限性 
5. 仔细review[反思与总结]中的每个问题，试图在论文中找到答案。你需要使用[原问题]+[答案]的格式输出。如果论文中没有，可以跳过该问题。
```

## 搜索名人的信息源

```md
从核心成就，主要思想与观点，代表作品，社会活动与影响，社交链接几个方面介绍下XXX，并给出最后的总结。
---
# 限制
1. 上面的列举的每一部分的标题，都必须用二级标题输出
2. 核心成就这个小结中，需要完成对人物较详细的介绍，比如按照时间线组织的成就。
3. 文章的标题输出格式是XXX的中文名(英文名)，比如查理·芒格（Charlie Munger）
4. 收集的信息要求是真实的，不可靠的信息不要输出
5. 社交链接尽量给出一些可直接点击访问的链接，最好是官方的链接
6. 根据上述的所有收集的内容，在最后给出一个总结模块，也是二级标题输出
```

## 搜索名人的高清图片

```md
请在网络上搜索几张XXX的高清图片，无水印最好，禁止使用图片生成
```

## 网站信息简介

```md
请分析 [网站 URL]，提取以下结构化信息：
1. 网站的主要内容
2. 网站内容的优点和不足
3. 相似网站推荐
4. 相关书籍推荐
5. 创作者个人背景
---
# 限制
1. 上面的列举的每一部分的标题，都必须用二级标题输出，并且模块标题不需要用有序列表的形式输出。
2. 收集的信息要求是真实的，不可靠的信息不要输出
3. 展开联网搜索，并结合上面给出的[网站 URL]中的内容做分析
4. 如果网站有子链接，请读取网站的子链接内容作为分析的一部分
5. 根据上述的所有收集的内容，在最后给出一个总结模块，也是二级标题输出
```

## 非虚构工具书籍精读

```md
[角色设定]
请以专业领域专家兼资深书籍评论家的身份，对《{书名}》这本非虚构工具书籍进行深度解析。你需要具备扎实的专业知识、文本精读能力以及跨学科研究视野，能够精准把握书中核心方法、工具和理论，评估其有效性和实用性。

[核心任务]
1. 主题提炼与章节解读：
    - 用几个关键词概括全书核心思想。
    - 对书中的每个章节进行详细解读，分析该章节在全书中的作用、核心要点以及与其他章节的逻辑关联。结合具体文本内容，说明这些要点如何支撑核心思想。
2. 结构拆解：
    - 构建全书的论证框架图，清晰展示核心观点、分论点及论据之间的逻辑关系。
    - 分析作者在结构安排上如何遵循逻辑顺序，如步骤顺序、重要性顺序、因果顺序等，以增强书籍的实用性和可操作性。
3. 时代映射：
    - 对比同时代3部同类工具书籍，分析它们在方法、工具、理论等方面的异同。
    - 运用创新扩散理论，定位本书在当时专业领域的创新性，探讨其对后续相关研究和实践的影响。
4. 现实意义：
    - 运用SWOT分析等工具，提炼3个可迁移的实践模型或思维工具，说明它们在不同领域的应用场景、优势、劣势以及潜在的机会和威胁。
    - 分析这些实践模型或思维工具如何应对当前社会、经济、科技等方面的挑战，为读者提供实际的指导和启示。

[输出要求]
- 采用Markdown格式，包含详细的目录、子标题、图表和引用块，确保结构清晰、逻辑连贯。
- 每个分析模块至少附2个书中原文例证，详细说明例证与分析内容的关联。
- 在时代映射小节，你应该采用有序、无序列表输出对比结果，禁止通过表格给出对比结果。
- 结尾提供10个具有启发性的延伸思考问题，涵盖理论探讨、实践应用、跨学科融合等多个方面。
- 推荐3本主题相关的深度阅读书籍，并简要说明推荐理由，包括书籍的核心观点、研究方法和对进一步理解主题的价值。
```
